To organize the input files, please follow these steps:

1. Download the files "cases-brazil-cities-time_2*.csv.gz" from the GitHub repository available at: https://github.com/wcota/covid19br. After downloading, move these files to the "data/input/covid" folder.

2. Next, download the file from the following link: https://www.gstatic.com/covid19/mobility/Region_Mobility_Report_CSVs.zip. Extract the contents of the ZIP file and locate the files named "202*_BR_Region_Mobility_Report.csv". Move these files to the "data/input/mobility" folder.

By following these steps, you will have organized the input files in their respective folders, ready for further processing.
