To efficiently organize your input files, please adhere to the following guidelines:

**Step 1: Download and Organize COVID-19 Data from SUS Monitoring Panel**

1. Visit the Painel Coronavirus website at https://covid.saude.gov.br/ to access the required data files.
2. Locate the button "Arquivo CSV".
3. Download the compressed file and decompress it in folder named "data/input/covid_monitoring_panel".

**Step 2: Download and Organize COVID-19 Data from SUS Mortality System**

1. Visit the data repository at https://opendatasus.saude.gov.br/fa_IR/dataset/sim to access the required data files.
2. Download the files referent the year 2020, 2021, and 2022 and put in data/input/covid_sim.

**Step 3: Download and Organize COVID-19 Data from Severe Acute Respiratory Syndrome provided by SUS**

1. Visit the data repository at https://opendatasus.saude.gov.br/dataset/srag-2020 and https://opendatasus.saude.gov.br/dataset/srag-2021-a-2023 to access the required data files.
2. Download the files referent the year 2020, 2021, and 2022 and put in data/input/covid_srag.

**Step 4: Obtain and Arrange Mobility Data**

1. Access the mobility data files via the link: https://www.gstatic.com/covid19/mobility/Region_Mobility_Report_CSVs.zip.
2. Extract the contents of the ZIP file to reveal files named "202*_BR_Region_Mobility_Report.csv".
3. Move these mobility data files to a dedicated folder named "data/input/mobility".

**Step 5: COVID-19 Data about Many Countries**

1. Download the desired file from the following link: https://github.com/owid/covid-19-data/blob/master/public/data/owid-covid-data.csv.
2. Place this file in the specified directory labeled "data/input/OurWorldData".

Upon meticulously following these steps, your input files will be systematically organized within their respective folders, all set for further processing and analysis. This organized structure will streamline your workflow and contribute to a smoother data processing experience.
